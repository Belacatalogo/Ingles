🚫 Popup bloqueado

O seu browser bloqueou a janela de login do Google. Veja como resolver:

  1. Chrome/Android: Toque no ícone de cadeado 🔒 na barra de endereço → Permissões → Popups → Permitir
  2. Safari/iOS: Configurações → Safari → Bloqueador de pop-ups → desativar para este site
  3. Alternativa: Acesse o app pelo Chrome no computador — o login funciona sem popup

Após permitir popups, toque em "Entrar com Google" novamente.
